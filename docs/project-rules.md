# 剧本创作规范

## 资源标记体系

### CG_OFFICIAL
官方CG原版对白或旁白。

### INGAME_OFFICIAL
游戏任务、事件、副本中的原版对白。

### BOOK_OFFICIAL
官方小说、编年史、设定集内容。

### CINEMATIC_RECREATION
官方设定存在，但没有对应影视资源，需要重新制作。

### ORIGINAL_NARRATION
原创旁白。

### ORIGINAL_DIALOG
原创对白。

### ASSET_REQUIRED
未来需要制作的素材。

---

## 创作原则

1. 官方设定优先
2. 官方对白优先
3. 不主动修改主时间线
4. 原创内容仅用于补充空白
5. 每集必须统计官方与原创占比
