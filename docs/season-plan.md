# 季度规划

## 创作原则

每一季代表一个历史纪元。每一季内部按“主线事件 + 地图章节 + 副本/团本故事 + 季终Boss”组织。

## 第一季：宇宙诞生

主题：一切战争的起源。

预计 8 集。

| 集数 | 标题 | 核心内容 | 官方资源占比 |
|---|---|---|---|
| S01E01 | 混沌之前 | 光与虚空诞生，现实宇宙形成 | 低 |
| S01E02 | 阿曼苏尔 | 第一位泰坦觉醒，寻找同伴 | 低 |
| S01E03 | 万神殿 | 泰坦集结，秩序理念形成 | 低 |
| S01E04 | 堕落者 | 萨格拉斯登场，开始猎杀恶魔 | 中 |
| S01E05 | 虚空之影 | 萨格拉斯发现被腐化的世界之魂 | 低 |
| S01E06 | 破碎信念 | 萨格拉斯与万神殿理念冲突 | 低 |
| S01E07 | 燃烧军团 | 萨格拉斯堕落，恶魔军团形成 | 中 |
| S01E08 | 艾泽拉斯 | 古神落入艾泽拉斯，泰坦远望新世界 | 低 |

## 第二季：泰坦降临

主题：泰坦与古神争夺艾泽拉斯。

重点内容：黑暗帝国、泰坦守护者、亚煞极、奥杜尔体系、守护巨龙前史。

## 第三季：暗夜帝国

主题：最辉煌的文明走向毁灭。

重点内容：艾萨拉、上层精灵、萨维斯、永恒之井、燃烧军团低语。

## 第四季：上古之战

主题：世界毁灭之日。

重点内容：军团第一次入侵、半神之战、巨龙之魂、永恒之井爆炸、大分裂。
