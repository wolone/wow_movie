# 第一季：宇宙诞生

## 季度定位

第一季讲述《魔兽世界》宇宙最初的历史：光与虚空、现实宇宙、虚空领主、泰坦觉醒、万神殿形成以及萨格拉斯堕落的前奏。

这一季没有玩家熟悉的联盟、部落、人类、兽人或暗夜精灵，因此采用“史诗旁白 + 泰坦视角 + 宇宙级视觉叙事”的方式。
